║  PRO TIPS                                                                    ║
║  ───────────────────────────────────────────────────────────────────────────║
║   • Mouse mode enabled - Click to switch panes/windows                       ║
║   • Ctrl+b z to zoom/unzoom current pane (great for focus)                   ║
║   • Ctrl+b : to enter command mode for advanced operations                   ║
║   • Use Ctrl+b [ then / to search through scrollback history                 ║
║   • Ctrl+b Ctrl+arrow for pane resizing (hold Ctrl+b)                        ║
║   • Ctrl+b ! to break pane into new window                                   ║
║   • Ctrl+b Ctrl+o to rotate panes                                            ║
║   • Use Ctrl+b w for visual window/session navigation                        ║

║      Win 0 → Edit src/array.cc                                               ║

║      Win 2 → git status && git diff                                          ║
║              git add . && git commit                                         ║
║                                                                              ║
║   3. Quick Prototyping (Quick Mode):                                         ║
║      Left pane  → Edit src/main.cc                                           ║
║      Right pane → cmake --build build && ./build/purgatory                   ║
║                   Iterate rapidly                                            ║
║                                                                              ║
║   4. Test-Driven Development:                                                ║
║      Win 0 → Write test in test/test.cc                                      ║
║      Win 1 → cmake --build build                                             ║
║              ctest --test-dir build --output-on-failure                      ║
║      Win 0 → Implement in src/*.cc until test passes                         ║
║      Win 2 → git commit with passing tests                                   ║
